# Popis:
Desktopová aplikace slouží jako uživatelské rozhraní pro monitorování a řízení domácích zařízení připojených k systémům jako Arduino. Umožňuje manuální zapnutí či vypnutí světel, sledování teploty a vlhkosti, nastavení režimů a aktivaci bezpečnostních funkcí. Aplikace může komunikovat přes sériové rozhraní, Bluetooth nebo síťové protokoly a poskytuje vizualizaci dat, historii událostí a konfigurační možnosti. Je důležité zajistit jednoduchost, stabilitu a zabezpečení spojení mezi aplikací a řídící jednotkou.
