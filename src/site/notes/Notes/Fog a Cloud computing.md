# Popis:
Fog a cloud computing poskytují distribuované výpočetní zdroje pro zpracování dat v IoT a chytrých domech. Edge/fog vrstvy umisťují výpočet blíže k senzorům a uživatelům, snižují latenci a zátěž sítě, zatímco cloud nabízí škálovatelné úložiště a výpočet pro náročné modely. Kombinace umožňuje rychlé vyhodnocení alarmů lokálně a souběžné trénování nebo archivaci v cloudu. Důležitá jsou rozhodnutí o tom, která data zpracovat lokálně kvůli soukromí a latenci a která posílat do vzdálených center.
