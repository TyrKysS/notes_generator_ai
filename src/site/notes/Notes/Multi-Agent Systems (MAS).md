# Popis:
Systémy vícera agentů jsou architektury, kde více autonomních agentů spolupracuje nebo soupeří v sdíleném prostředí. MAS zkoumají, jak distribuované rozhodování, komunikace, delegace cílů a koordinace vedou k kolektivní inteligenci. Tyto přístupy modelují sociální interakce, rozdělení práce a škálovatelné řešení problémů, přičemž využívají principy autonomie, percepce a nastavování strategií pro dosažení komplexních cílů.
