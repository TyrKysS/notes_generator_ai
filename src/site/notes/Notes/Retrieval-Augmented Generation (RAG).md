# Popis:
RAG kombinuje vyhledávání relevantních dokumentů z externích úložišť s generativním modelem, který použije nalezené informace ke konstruování odpovědí. Tím se zvyšuje přesnost, factualita a schopnost pracovat s aktuálními nebo specializovanými znalostmi, které nemusí být obsaženy v parametrech modelu. RAG slouží jako klíčový mechanismus pro doplnění paměti a kontextu u agentních systémů a snižuje problémy spojené s halucinacemi.
